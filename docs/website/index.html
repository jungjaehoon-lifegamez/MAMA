What is MAMA?

An always-on AI companion that remembers your decisions and works across all your platforms.

Traditional AI Forgets everything after each session
MAMA Remembers decisions, tracks evolution, works 24/7 from Discord/Slack/Telegram
  • Run as Discord, Slack, or Telegram bot
  • Chat with voice from mobile (Web Speech API)
  • Visualize decision evolution in graph viewer
  • Create custom skills with Skill Forge
  • Schedule tasks with cron jobs

Choose Your Package

@jungjaehoon/mama-os

Standalone Agent

Your AI Operating System. Always-on agent with Discord, Slack, Telegram bots and web dashboard.

  • Gateway integrations (Discord, Slack, Telegram)
  • MAMA OS web interface
  • Mobile chat with voice
  • Skill Forge for custom skills
  • Cron scheduler
npm install -g @jungjaehoon/mama-os
mama init && mama start

@jungjaehoon/mama-server

MCP Server

Memory server for Claude Code and Claude Desktop. Decision tracking via MCP tools.

  • 4 MCP tools (save, search, update, load_checkpoint)
  • Semantic search with embeddings
  • Decision graph with edge types
  • Works with any MCP client
npx @jungjaehoon/mama-server

Claude Code Plugin

Plugin

Auto-context injection for Claude Code. Relevant decisions surface automatically.

  • /mama-* slash commands
  • Auto-recall hooks
  • Context injection on every prompt
/plugin marketplace add jungjaehoon-lifegamez/claude-plugins
/plugin install mama

Key Features

Gateway Integrations

Run MAMA as a Discord, Slack, or Telegram bot. Chat with your AI from anywhere, on any device.

Discord | Slack | Telegram

Mobile Chat

Voice input with Web Speech API. Text-to-speech responses. Hands-free mode for cooking or driving.

Decision Evolution

Track how decisions evolve. See what worked, what failed, and why your thinking changed.

Semantic Search

Natural language queries find relevant decisions. Multilingual support (Korean + English).

Skill Forge

Create custom skills with 3 AI agents (Architect, Developer, QA). Extend MAMA's capabilities.

Cron Scheduler

Schedule automated tasks. Daily briefings, weekly reports, or any recurring task.

Quick Install

Choose your preferred setup

MAMA OS (Standalone)

Recommended

Always-on AI agent with Discord/Slack/Telegram bots and web dashboard. 

# Authenticate Claude CLI first
claude

# Install and start MAMA
npm install -g @jungjaehoon/mama-os
mama init
mama start

Access dashboard at http://localhost:3847

Claude Code Plugin

Plugin with slash commands and auto-context hooks.

# Add marketplace (one-time)
/plugin marketplace add jungjaehoon-lifegamez/claude-plugins

# Install plugin
/plugin install mama

Claude Desktop (MCP)

MCP server for Claude Desktop app.

{
  "mcpServers": {
    "mama": {
      "command": "npx",
      "args": ["-y", "@jungjaehoon/mama-server"]
    }
  }
}

Add to claude_desktop_config.json

Prerequisites

  • Node.js >= 22.0.0
  • Claude CLI authenticated (claude)
  • 500MB free disk space
